## 🚀 Getting Started

To run the project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to see the magic! 🌐✨

You can start editing the main page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

---

## 🧭 Guidance

1. **Clone the repository:**
	```bash
	git clone https://github.com/olivierlarose/3d-earth-scroll.git
	cd 3d-earth-scroll
	```
2. **Install dependencies:**
	```bash
	npm install
	```
3. **Run locally:**
	```bash
	npm run dev
	```
4. **Explore and customize:**
	- Main code: `src/app/page.js`
	- 3D Earth: `src/components/earth/`
	- Smooth scroll: `src/components/smoothScroll/`
